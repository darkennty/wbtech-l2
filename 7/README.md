## Задача

Что выведет программа?

Объяснить работу конвейера с использованием select.

```
package main

import (
	"fmt"
	"math/rand"
	"time"
)

func asChan(vs ...int) <-chan int {
	c := make(chan int)
	go func() {
		for _, v := range vs {
			c <- v
			time.Sleep(time.Duration(rand.Intn(1000)) * time.Millisecond)
		}
		close(c)
	}()
	return c
}

func merge(a, b <-chan int) <-chan int {
	c := make(chan int)
	go func() {
		for {
			select {
			case v, ok := <-a:
				if ok {
					c <- v
				} else {
					a = nil
				}
			case v, ok := <-b:
				if ok {
					c <- v
				} else {
					b = nil
				}
			}
			if a == nil && b == nil {
				close(c)
				return
			}
		}
	}()
	return c
}

func main() {
	rand.Seed(time.Now().Unix())
	a := asChan(1, 3, 5, 7)
	b := asChan(2, 4, 6, 8)
	c := merge(a, b)
	for v := range c {
		fmt.Print(v)
	}
}

```

## Решение

Данная программа ведёт работу с каналами на запись. С помощью функции `asChan()` создаются два канала `a` и `b`, которые объединяются в один канал `c`, используя функцию `merge()`.

Функция `asChan()` принимает на вход слайс со значениями, которые должны быть записаны в новый канал, и возвращает этот канал. В функции инициализируется горутина для записи значений из переданного слайса в канал, после чего канал закрывается.

Функция `merge()` принимает два канала, которые необходимо объединить, и создаёт новый канал, в который с помощью оператора `select` записываются данные по мере их считывания из объединяемых каналов. Когда из обоих переданных каналов будут прочитаны все данные, результирующий канал закроется.

После создания двух начальных и результирующего канала данные из последнего выводятся в консоль. Цикл считывания и вывода данных из объединённого канала завершится в тот момент, когда из него будут считаны все данные и канал будет закрыт.
